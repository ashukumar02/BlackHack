# Custom Bambu Lab Wireless Mouse

A custom 3D-printable mouse shell designed specifically for the official Bambu Lab Wireless Mouse Components Kit.

## Project Files

* **[`CAD/`](CAD/)** - Contains the `.stl` files for 3D printing and the source `.step` files if you want to modify the design.
* **[`Assets/`](Assets/)** - Contains images and photos of the completed mouse.

## What You Need

* 1x Bambu Lab Wireless Mouse Components Kit 002
* 1x AA Battery
* 3D Printer and filament (PLA or PETG recommended)
